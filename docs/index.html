TWW AP Randomizer Tracker

This is a tracker for The Wind Waker Archipelago Randomizer. If you want to run the tracker locally, please follow the steps in the Build Instructions section in order to get started. otherwise, you may use the tracker by refering here.

Build Instructions

Building and running the tracker locally requires you to install Node.JS Version 20 or newer and Git.

Clone the repository by running the following in a command prompt:

git clone https://github.com/josephanimate2021/twwr-ap-tracker.git

Navigate to the twwr-ap-tracker folder and install dependencies:

cd twwr-ap-tracker && npm install

You can then build and serve the tracker application:

npm start

After the server starts, you can go to localhost:8080 to open the tracker.

I heard that this source code contains the old branch which has working AP Support and the older styled tracker. is that true?

Yes, there is an older style tracker that you may use if you wish. Both trackers work in a simmilar structure, but with different code. If you prefer to use the older style tracker than the newest style one, you may click here to get started. Please keep in mind that the older style tracker supports up to version 1.9.0 of The Wind Waker Randomizer, meaning that the older style tracker may not understand how the WWR logic works in AP. but it's support still works nonetheless. (Only downside is that you won't be able to copy permalinks from the applied AP Settings as the code for the older tracker has no idea how to create new permalinks.)
