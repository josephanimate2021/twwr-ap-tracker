TWW AP Randomizer Tracker

This is a tracker for The Wind Waker Archipelago Randomizer. For usage of this newer style tracker, you may run this locally as I can't get this branch to deploy with gh pages for some reason.

Build Instructions

Building and running the tracker locally requires you to install Node 20 and Git.

Clone the repository by running the following in a command prompt:

git clone https://github.com/josephanimate2021/twwr-ap-tracker.git

Navigate to the twwr-ap-tracker folder and install dependencies:

cd twwr-ap-tracker && npm install

You can then build and serve the tracker application:

npm start

After the server starts, you can go to localhost:8080 to open the tracker.
